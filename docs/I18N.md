# 多言語（Phase 4）

Daily Three は **日本語（`ja`）** と **英語（`en`）** の2言語です。

## URL

| 言語 | 例 |
|------|-----|
| 英語（デフォルト） | `https://MTNT-Products.github.io/daily-three/en/` |
| 日本語 | `https://MTNT-Products.github.io/daily-three/ja/` |
| digest | `.../ja/digest/2026-05-27/`、`.../en/digest/2026-05-27/` |

ルート `/daily-three/` は **初回のみ** 言語を選んでリダイレクトします。

- **デフォルト**: 英語（`/en/`）
- **ブラウザの優先言語が日本語**（`ja`, `ja-JP` など）: `/ja/`
- **2回目以降**: `localStorage` の `dt_lang`（言語切替で保存）

## digest コンテンツ

- 日本語: `src/content/digest/ja/YYYY-MM-DD.md`
- 英語: `src/content/digest/en/YYYY-MM-DD.md`
- CI の digest 生成は **1回の LLM 呼び出しで ja+en を同時出力**（英語は翻訳ではなく英語プロンプト要約）
- **英語 MD が無い日**は `/en/digest/...` は **404**（日本語への自動フォールバックなし）

## 旧 URL

`/daily-three/digest/YYYY-MM-DD/` は `localStorage` または **日本語** へリダイレクトする静的ページを残しています。

## UI

- 全ページの **ヘッダー右上** に言語切替（日本語 / English）。同じパスで `ja` ↔ `en` に遷移し、`localStorage` の `dt_lang` を更新します。

## 開発

```bash
npm run dev
# http://localhost:4321/daily-three/en/
# http://localhost:4321/daily-three/ja/
```

UI 文言: [`src/i18n/ui.ts`](../src/i18n/ui.ts)
