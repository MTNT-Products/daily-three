# Daily Three — UIモック（方式A）

## 見方

1. ターミナルでプロジェクトルートから:

```bash
npx serve mockups
```

2. ブラウザで `http://localhost:3000/compare.html` を開く（ポートは表示に従う）

3. **横スクロール**で案A〜Dを並べて比較する

`compare.html` を直接ダブルクリックしても表示できますが、一部ブラウザでは画像の読み込みに制限がある場合があります。

## ファイル

| ファイル | 役割 |
|----------|------|
| `compare.html` | 4案を横並びで比較 |
| `shared/dummy-data.json` | 全案共通のダミー文言（編集用） |

## 選び方

気に入った案をチャットで伝える例:

- `案Bで進めて`
- `案Cの画像比率 + 案Aの見出しフォント`

## スキル

今後同じ手順でUI比較するときは Cursor で `@ui-mockup-compare` を指定してください。
